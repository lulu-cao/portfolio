Kansas City International Airport Content Management System

This is my biggest project at Dimensional Innovations, where I leveraged multiple tech stacks to create and deploy a content management system (CMS) for the Kansas City International Airport. It stores data and provides API for at least 5 frontend apps. During the development process, I also worked on our windows service scripts that interact with REST APIs from green hippo for media playing.

Highlight: Half of this project involves writing up express routers to migrate all data from our previous Django CMS to this new CMS and serve data in the same data structure for frontend apps.

AdminJS, ExpressJS, TypeORM, Postgres, ReactJS, Django, Green Hippo, Docker

Durham Bulls Exhibit

This is my very first project at Dimensional Innovations, where I created and deployed 2 kiosk apps that are currently on display at Durham Bulls' Baseball Club History & Storytelling Exhibit. This app is one of them. The app starts with an attract loop and follows with introduction, video, and image galleries.

Highlight: I created the slide carousel from scratch, which demonstrates my competency for complex calculation and analytical thinking.

VueJS, ElectronJS, Tailwind, CSS

St. Louis Cardinals Hall of Fame and Cardinal Nations

This is one of my most recent projects where I created and deployed three kiosk apps that will be displayed in St. Louis Cardinals Hall of Fame and Museum. Two of the apps are a pair that communicate with each other using websocket to switch maps. The display app displays choropleth maps using LeafletJS and geojson. The control app controls map switching on the display app, allows users to geo search their origin locations, and stores data in our content management system. The keyboard is implemented with the simple-keyboard package.

VueJS, LeafletJS, Simple Keyboard, SwiperJS, WebSocket, ElectronJS

App Restart Service

This is a simple tool for users to monitors various system parameters on a remote PC, and for them to restart any .exe application on that PC. I created a Node.js application and compiled it into a .exe file with nexe package. I added handlers in our content management system for users to send commands to restart desired apps. And I used WebSocket to communicate with servers and send commands to kill processes when requested.

NodeJS, WebSocket, AdminJS, Docker

Mobile Donation Web App

This is a simple web app created for prototyping and concept demonstration. It allows mobile users to choose a donation amount and launch a critter on donor walls. The kind of critter rewarded depends on users' donation amount. It also connects with a Django CMS to store related data and allows users to search their past donations. The final app is a collaborative effort between the developers at Children's Healthcare of Atlanta and me. You can view it here. The part I worked on includes the search page and the thank you page.

Highlight: The prototyping project is the first project I have completed from the very beginning to the very end all by myself, from creating AWS buckets and IAM users to setting up GitLab repositories and Docker containerized Django CMS and API, to coding and deploying the mobile website on Heroku.

VueJS, Django, Postgres, Heroku, AWS, Docker

Biopreparedness Exhibit

A simple kiosk app that displays video galleries, image galleries, timelines, and third-party mini-games. This app is currently displayed at the Wittson Hall in the University of Nebraska Medical Center, educating the public about biopreparedness.

VueJS, ElectronJS, Tailwind, CSS
